# Семейный бюджет

Простое веб-приложение для ведения семейного бюджета на Flask.

# Как запустить

1. Устанавливаем зависимости
```bash
py -m pip install -r requirements.txt
```
2. Запускаем приложение
```bash
py app_simple.py
```

ктрл+лкм или в браузере открываем `http://localhost:5000`

Когда закончишь работать, нажми Ctrl+C чтобы остановить сервер.

#Что работает

доходы и расходы
текущий баланс

поиск транзакций по описанию
бэкапы базы данных

База данных (`family_budget.db`) уже есть в проекте.
